Windows软件推荐

发表于2020-11-29|更新于2022-06-19|笔记

压缩

Bandizip

推荐理由：Bandizip 是一个强大的多功能解压缩文件管理工具，它提供了高速的解压缩速度和其他各种实用功能
备注：推荐6.29版本，没有广告,有便携版

文档

Microsoft Office

推荐理由：微软的生产力套件

Sumatra PDF

推荐理由：一款Pdf，epub，Mobi，Chm， Xps， Djvu，Cbz，Cbr阅读器
备注：有便携版

Calibre

推荐理由：一个强大和易于使用的电子书管理器
备注：有便携版

下载

Motrix

推荐理由：一款全能的下载工具支持下载 HTTP、FTP、BT、磁力链接等资源

图形

Snipaste

推荐理由：是一个简单但强大的截图工具

Screentogif

推荐理由：它允许你录制屏幕的一部分区域并保存为 gif 或视频

Photoshop

推荐理由：一款优秀的音像处理软件

视频

Mpv

推荐理由：免费、开源和跨平台媒体播放器

premiere

推荐理由：一款非线性编辑的影片剪辑软件

云储存

OneDrive

推荐理由：为Windows用户提供最佳云存储解决方案

开发者工具

FileZilla

推荐理由：FileZilla客户端不仅支持 FTP，还支持 TLS （FTPS） 和 SFTP 上的 FTP

Visual Studio Code

推荐理由：轻量级源代码编辑器

Mitmproxy

推荐理由：是一个免费和开源的交互式 HTTPS 代理

其它

VMware Workstation Pro

推荐理由：强大的虚拟机软件

MindMaster

推荐理由：思维导图

CopyTranslator

推荐理由：复制即翻译的外文辅助阅读翻译解决方案

PicGo

推荐理由：一个用于快速上传图片并获取图片 URL 链接的工具
